# URL-Manipulator 
A simple python3 script that let's you shorten, expand or check the number of clicks your link has in your local environment

#### Functionalities
 - url-shortner
 - url-expander
 - url-clicks-counter

## Installation
##### Cloning
```
$ git clone https://github.com/Reechychukz/url-manipulator.git
$ cd url-manipulator

```

## Usage
...
To shorten URL
$ python UrlShortner
